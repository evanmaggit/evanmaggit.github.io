What do these CSS properties do?

"color" sets the color of the text inside an element.

"background-color" fills the background area of an element with color, behind both the text and padding area.

"border-color" defines the color of an element’s border.

How does CSS accept color as a value?

CSS includes a library of predefined color names that are simple, readable, and great for quick use.

Hex codes use a # followed by either three or six hexadecimal digits.

RGB stands for Red, Green, Blue. You can also add an “A” for alpha (opacity), ranging from 0 to 1.

Another color model — Hue, Saturation, Lightness (HSL) — (Hue is a degree on the color wheel (0–360), saturation and lightness are percentages).
